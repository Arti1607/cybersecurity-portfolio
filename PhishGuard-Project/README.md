# 🛡️ PhishGuard  
## Web-Based Phishing Detection and Cybersecurity Awareness System

PhishGuard is a cybersecurity-based web application developed to help users identify suspicious URLs, improve cybersecurity awareness, and understand phishing attacks through educational content and interactive quizzes.

The system performs basic phishing URL analysis using security rules and generates risk-based results for users. It also provides awareness modules, quiz functionality, user history tracking, and downloadable PDF security reports.

---

# 📌 Project Objective

The main objective of this project is to:

- Detect suspicious and phishing URLs
- Create cybersecurity awareness among users
- Educate beginners about online threats
- Encourage safe browsing practices
- Provide an easy-to-use phishing analysis platform

---

# 🚨 Problem Statement

Many internet users are unaware of phishing attacks and fake websites.  
Cybercriminals use malicious links, fake login pages, and social engineering techniques to steal sensitive information such as:

- Passwords
- OTPs
- Banking details
- Personal information

Existing cybersecurity tools are often complex for beginners.  
PhishGuard was developed as a simple educational phishing detection platform for students and normal internet users.

# 🔐 Security Rules Used

The phishing detection module checks for:

- Presence of '@' symbol
- Long suspicious URLs
- Missing HTTPS protocol
- Suspicious keywords
- Blacklisted domains

---

# ⚠️ Limitations

- Rule-based detection only
- No machine learning integration
- Requires internet/browser environment
- Limited blacklist database
- Localhost project (not deployed on live server)
